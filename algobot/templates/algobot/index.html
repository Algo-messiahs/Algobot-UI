About The Project

Built using Alpaca API & Python

Our Algorithmic trading bot is created using Alpaca API and python. Our bot has the ability to generate and execute buy and sell signals in financial markets. The main components of such a robot include entry rules that signal when to buy or sell, exit rules indicating when to close the current position, and position sizing rules defining the quantities to buy or sell.
